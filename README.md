# Helper Toolbox

Quality-of-life and productivity tooling for automotive testing, diagnostics,
and validation (Helper Toolbox + Testrun Manager).

This repository hosts **release binaries only** — installers and update
manifests. Source code is not published here.

## Download

Grab the latest installer from the [Releases](../../releases/latest) page:
`Setup_HelperToolbox_<version>.exe`.

The installer is per-user (no admin rights required) and installs both
Helper Toolbox and Testrun Manager side by side, with an in-app self-updater.
